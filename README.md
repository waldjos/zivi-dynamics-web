# Zivi Dynamics Web

Sitio corporativo multipágina de **Zivi Dynamics C.A.**

## Stack
- Next.js + TypeScript
- App Router
- CSS responsivo
- Framer Motion disponible para futuras transiciones
- Vercel

```bash
npm install
npm run dev
```

## Compilación

